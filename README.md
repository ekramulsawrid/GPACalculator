# GPACalculator

GPA Calculator is a android application. that calculates GPA based on user inputs. There are input validation so that invalid inputs from users are denied.

## Author

Ekramul Sawrid

## Operating System

Android (Mobile)

## Language

Java, FXML

## Features

Input Validation
